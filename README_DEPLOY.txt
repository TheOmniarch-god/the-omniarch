THE OMNIARCH — LIVE DEPLOY PACKAGE

Main file:
- index.html

Best fast deployment options:

1) Netlify Drop
- Go to https://app.netlify.com/drop
- Drag the entire omniarch-live folder into the page.
- Netlify gives you a temporary live URL immediately.
- Later, connect the domain theomniarch.com.ng in Site settings > Domain management.

2) Vercel
- Create a new project.
- Upload/import this folder.
- Framework preset: Other / Static.
- Build command: none.
- Output directory: leave blank or ./

3) cPanel / existing hosting
- Open File Manager.
- Go to public_html for the domain.
- Upload index.html, robots.txt, sitemap.xml, and _redirects if supported.
- If replacing an old site, back up the old files first.

4) GitHub Pages
- Put these files in a GitHub repository.
- Enable Pages from the main branch/root.
- Add custom domain later if needed.

Important:
- This is a static website. No backend required.
- External live YouTube thumbnails may be blocked in some previews/sandboxes but should work better on a normal deployed domain.
